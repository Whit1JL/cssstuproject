CSS Snippet Cheatsheet

Pssttt! I know they say cheating is bad, but we still created a CSS cheatsheet for you. See below for some very useful (and cool!) CSS tricks... [Hover Over Me]

Hover Selector

This will select elements when you place the mouse over it. 

         
         a:hover {
            background-color: purple;
         }

Padding

Utilize this to create space around an element's content.

         
         div {
            padding-left: 2px;
         }

Text Transform

This will change the text.

         
         div.c {
            text-transform: lowercase;
         }

Pseudo Class

Use this to define a special state of an element.

         
         selector:pseudo-class {
           property: value;
         }

Variables

This will reuse specific values throughtout a document.

         
         var(--name)

Media Query

Use this @media rule to include a block of CSS properties only if a certain condition is true.

         
         @media only screen and (max-width: 90px) {
         body {
            background-color: blue;
         }
         }
